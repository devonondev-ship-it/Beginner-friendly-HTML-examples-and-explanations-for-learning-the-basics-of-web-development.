# HTML for Beginners

A simple collection of HTML examples and explanations for beginners.

This repository was created to help friends and new developers understand the fundamentals of HTML step by step.

The examples are intentionally simple and beginner-friendly.

---

## 📚 Topics Covered

1. HTML Structure
2. Headings and Text
3. Links
4. Images
5. Lists
6. Div Containers
7. Class and ID
8. Anchor Links (Back to Top)
9. Span
10. Buttons
11. Forms
12. Style Tag
13. Semantic HTML Tags

---

## 🎯 Goal

The goal of this repository is to make HTML easier to understand by providing:

- simple explanations
- clean example code
- beginner-friendly structure

---

## 📂 Structure

Each folder contains one topic with a simple HTML example.

Example: 

01-html-structure/
index.html

02-headings-and-text/
index.html

03-links/
index.html


---

## 🚀 Who is this for?

This project is for:

- complete beginners
- friends learning web development
- anyone who wants simple HTML examples

---

## 💡 Tip

The best way to learn HTML is to:

1. Read the example
2. Run the code
3. Experiment and change things
4. Try building small pages
